# 图片压缩工具

一个简洁优雅的在线图片压缩工具,采用苹果风格设计,帮助用户轻松压缩图片文件大小。

## 功能特点

- 支持上传 PNG、JPG、JPEG 等常见图片格式
- 可自定义压缩比例(0-100%)
- 实时预览压缩前后的图片效果
- 显示压缩前后文件大小对比
- 一键下载压缩后的图片
- 响应式设计,支持各种设备访问
